## Compilation

- ### Ant
  Run `ant build.xml`
  
- ### Java Compiler
 Â Run `javac Fractal.java -d .`
  
## Running
  Run with :
  - `java Fractal`
  - `java Fractal [Mandelbrot|Julia]`
  - `java Fractal Mandelbrot [realmin, realmax, imagmin, imagmax] [iterations]`
  - `java Fractal Julia [real imag] [iterations]`
